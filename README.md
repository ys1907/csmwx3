# 到底吃点啥 · 情侣版 🍽️

一款帮情侣 / 朋友解决「今天到底吃什么」这个世纪难题的微信小程序。内置 500+ 道菜的数据库，提供转盘抽签、塔罗食运、默契 PK 等多种趣味决策方式。

## ✨ 功能特性

- **🎰 转盘抽签** —— 随机为你选出今天的那一餐，告别选择困难。
- **🔮 塔罗食运** —— 每日抽取一张「食运」，根据运势主题（烈焰 / 月光 / 海洋……）推荐对应口味的菜。
- **⚔️ 默契 PK** —— 两人同时选口味，测测彼此的默契值，附带请客 / 洗碗等趣味「奖惩」。
- **📅 每周主题** —— 周一到周日各有主题氛围，给日常加点仪式感。
- **❤️ 收藏与历史** —— 记录吃过的、收藏喜欢的，数据本地保存。
- **🍳 食物管理** —— 自由增删改菜品，按分类、场景、预算、辣度等维度管理。

## 📂 项目结构

```
csmwx3/

```

## 🚀 本地运行

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. 打开微信开发者工具，选择 **导入项目**，目录指向本仓库根目录。
3. 填入你自己的小程序 AppID（或选择「测试号」）。
4. 点击 **编译**，即可在模拟器中预览。

## 💾 数据存储

所有用户数据（菜品、历史、收藏、PK 记录）均通过微信小程序的本地缓存 `wx.setStorage` 保存在设备本地，**不上传任何服务器**。存储键带版本号前缀（如 `wtec_foods_v3`），便于后续数据结构升级与迁移。

## 🛠️ 技术栈

- 原生微信小程序框架（WXML / WXSS / JavaScript）
- 无第三方依赖，开箱即用
